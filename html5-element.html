HTML Elements

Headings Paragraphs Links Lists Tables

HTML Headings

Headings are used to create titles or subtitles on a webpage. HTML offers six levels of headings, from <h1> to <h6>. The <h1> element defines the most important heading, and <h6> defines the least important one.

<h1>This is an H1 heading</h1>
<h2>This is an H2 heading</h2>
<h3>This is an H3 heading</h3>
<h4>This is an H4 heading</h4>
<h5>This is an H5 heading</h5>
<h6>This is an H6 heading</h6>

HTML Paragraphs

Paragraphs are used to define blocks of text in HTML. The <p> element is used to create a paragraph.

<p>This is a paragraph of text in HTML.</p>

HTML Links

Links are used to create hyperlinks in HTML. The <a> element is used to create a link, and the href attribute specifies the URL of the page the link goes to.

<a href="https://www.example.com">Visit Example.com</a>

HTML Lists

Lists are used to group related items. HTML supports ordered lists (<ol>) and unordered lists (<ul>), with list items defined using the <li> element.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

HTML Tables

Tables are used to display tabular data. The <table> element is used to create a table, and it contains rows (<tr>), headers (<th>), and data cells (<td>).

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
    <th>City</th>
  </tr>
  <tr>
    <td>John</td>
    <td>30</td>
    <td>New York</td>
  </tr>
  <tr>
    <td>Jane</td>
    <td>25</td>
    <td>Los Angeles</td>
  </tr>
</table>
