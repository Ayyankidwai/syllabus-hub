Introduction to CSS and CSS3

What is CSS?

CSS, or Cascading Style Sheets, is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS allows you to control the layout, colors, fonts, and other visual aspects of a webpage. Without CSS, webpages would be unstyled and difficult to read.

CSS vs CSS2 vs CSS3

CSS has evolved over time. CSS2 was released in 1998, and CSS3 was released in 1999. While CSS3 builds on the features of CSS and CSS2, it also includes new features and improvements.

  • Selectors: CSS3 introduced new selectors, such as the attribute selector, the nth-child selector, and the adjacent sibling selector, that allow you to select specific elements more easily.
  • Box Model: The box model determines how elements are sized and spaced on a webpage. CSS2 introduced a new box model which included padding, borders, and margins. CSS3 introduced the flexible box layout module, which allows for more flexible layouts.
  • Animations and Transitions: CSS3 introduced new animation and transition properties, such as transition, animation, and keyframes, which allow for more dynamic and engaging web designs.

New Features of CSS3

CSS3 brought a lot of new features and capabilities to web design, making it easier to create visually stunning and interactive web pages. Some of the new features include:

  • Combinator: CSS3 introduces a new general sibling combinator, which matches sibling elements using the tilde (~) combinator.
  • CSS Selectors: CSS3 selectors are more advanced than the simple selectors offered by CSS, providing a sequence of easy-to-use and simple selectors.
  • Pseudo-elements: CSS3 adds many new pseudo-elements for easier and more detailed styling. The new convention of double colons (::) is also introduced.
  • Border Style: CSS3 includes new border styling features like border-radius, border-image-slice, border-image-source, and values for “width stretch”.
  • Background Style Properties: CSS3 introduces new background style properties such as background-clip, background-size, background-style, and background-origin.
  • Flexbox: A new layout module that allows for more efficient and manageable layouts.
  • Grid: CSS Grid Layout allows web designers to create complex and responsive grid-based layouts with ease.
